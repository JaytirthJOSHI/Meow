MeowLang

A feline-friendly esoteric programming language where every command sounds like a cat!

Try It Now Install with pip

Why MeowLang?

Educational

Perfect for learning programming concepts in a fun, approachable way. Great for beginners and educators!

Feline-Friendly

Every command sounds like a cat! From 'meow' to 'purr', programming has never been this adorable.

Esoteric

Based on Brainfuck but with a cat-themed twist. Challenge yourself with this unique programming language!

Quick Start

Installation

pip install meowlang

Command Line

meow hello.meow
meow -c "meow meow meow purr"
