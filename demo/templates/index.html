MeowLang

A feline-friendly esoteric programming language where every command sounds like a cat!

Try It Now Install with pip

Why MeowLang?

Educational

Perfect for learning programming concepts in a fun, approachable way. Great for beginners and educators!

Feline-Friendly

Every command sounds like a cat! From 'meow' to 'purr', programming has never been this adorable.

Esoteric

Based on Brainfuck but with a cat-themed twist. Challenge yourself with this unique programming language!

Quick Start

Installation

pip install meowlang

Command Line

meow hello.meow
meow -c "meow meow meow purr"

Your First Program

🐾 Hello World in MeowLang
meow meow meow meow meow meow meow meow meow meow 🐾 Set cell 0 to 10
yowl 🐾 Start loop
    meow meow meow meow meow meow meow meow meow meow 🐾 Add 10 to cell 0
    right 🐾 Move to cell 1
    meow meow meow meow meow meow meow meow meow meow 🐾 Set cell 1 to 10
    left 🐾 Back to cell 0
    hiss 🐾 Decrement cell 0
paw 🐾 End loop
right 🐾 Move to cell 1
purr 🐾 Output cell 1 (should be 10)

Core Commands

meow

Increment current cell

hiss

Decrement current cell

purr

Output current cell

left/right

Move pointer

yowl/paw

Loop control

Fibonacci Sequence

🐾 Fibonacci in MeowLang
meow meow meow meow meow meow meow meow meow meow 🐾 Set cell 0 to 10 (counter)
yowl 🐾 Start loop
    right 🐾 Move to cell 1
    meow meow meow meow meow meow meow meow meow meow 🐾 Set cell 1 to 10
    right 🐾 Move to cell 2
    meow meow meow meow meow meow meow meow meow meow 🐾 Set cell 2 to 10
    left left 🐾 Back to cell 0
    hiss 🐾 Decrement counter
paw 🐾 End loop
right 🐾 Move to cell 1
purr 🐾 Output first number
right 🐾 Move to cell 2
purr 🐾 Output second number

Simple Calculator

🐾 Add two numbers
mew 🐾 Get first number
right 🐾 Move to next cell
mew 🐾 Get second number
left 🐾 Back to first cell
knead 🐾 Add cells together
