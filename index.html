Quick Start

Installation

pip install meowlang

Command Line

meow hello.meow
meow -c "meow meow meow purr"

Core Commands

meow - Increment current cell
hiss - Decrement current cell
purr - Output current cell
left/right - Move pointer
yowl/paw - Loop control
